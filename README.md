# Application Description

This experiment software contains 4 experiments designed to study human-computer interaction and user behavior.
Each experiment contains a different date picker and tracks user time and accuracy while completing date selection tasks.

# Installation Instructions

This software requires .NET Framework support and mulitplatform app development.

1. Clone the <a href="https://github.com/Dpastor1/CS464Proj">repository</a> to your local machine, or use the source code provided. 
2. Open the CS464Proj folder.
3. execute `dotnet run` in the terminal.

Alternatively, you can watch  <a href="https://youtu.be/UfpyOTNqSTo">this video</a> for more detailed installation instructions for interfacing through Visual Studio.
